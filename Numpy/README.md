## About NumPy

NumPy (Numerical Python) is the fundamental library for numerical computing in Python. It provides support for multidimensional arrays, mathematical operations, linear algebra, random number generation, and many other features required for AI and scientific computing.

Almost every major Python library used in AI—such as Pandas, Scikit-learn, TensorFlow, and PyTorch—builds upon NumPy.
